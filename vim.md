##### show line numbers
```
:set number
```
##### forward search for word
- navigate with n (forward), N (backward)
```
/word
```

##### backward search for word
- navigate with n (backward), N (forward)
```
?word
```

##### replace word1 with word2 in current line
- see https://linuxize.com/post/vim-find-replace/
```
:s/word1/word2/g
```

##### replace word1 with word2 in current line
- see https://linuxize.com/post/vim-find-replace/
```
:%s/word1/word2/g
```
